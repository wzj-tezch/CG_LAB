## 一、简答题（以下为全部题目作答）

### 1. 顶点插值法 vs 几何内在参数法

| | 顶点插值法 | 几何内在参数法 |
|---|-----------|---------------|
| **思想** | 在已知网格顶点间对位置/法向/颜色等做线性或样条插值 | 用物体自身的几何量（弧长、面积、测地线距离）作为参数 |
| **参数** | 时间 t 或关键帧参数 | 弧长 s、测地坐标等 |
| **优点** | 实现简单，适合网格动画 | 运动更自然，不受参数化影响（匀速沿测地线） |
| **典型** | 关键帧 + 线性/样条插值 | 测地线插值、内在形状混合 |

### 2. 基于特征的图像变形 vs 直接变形

**基于特征的变形**：用户指定线段、轮廓、对应特征，算法根据特征约束构造位移场再变形图像。

**优点**（相对直接逐像素拖拽）：

- 交互直观，只需控制少量特征；
- 整体变形协调，避免局部撕裂；
- 可保持线段长度/交叉关系等约束（如 Beier-Neely、Field 方法）。

### 3. 物理仿真计算的一般步骤（结合流程图）

```
1. 初始化物体状态（位置、速度、质量、约束等）
2. 循环（每一时间步 t^n -> t^{n+1}）：
   a. 根据当前状态计算力/力矩（重力、弹簧力、碰撞力等）
   b. 用数值积分器求解物理方程（如 F=ma）
   c. 更新状态（位置、速度）
   d. 处理约束与碰撞
   e. 渲染当前帧
3. 直到模拟结束
```

**核心**：物理方程 + 时间离散化（欧拉/Verlet/Runge-Kutta 等）。

### 4. 显式 / 隐式 / 半隐式欧拉法

| 方法 | 公式思想 | 特点 |
|------|---------|------|
| **显式 Euler** | `v_{n+1} = v_n + Δt·f(x_n)`，`x_{n+1} = x_n + Δt·v_{n+1}` | 实现简单；大步长易**不稳定**（弹簧系统易爆炸） |
| **隐式 Euler** | 用 `f(x_{n+1})` 更新 | **无条件稳定**（线性分析）；需求解非线性方程组，每步代价高 |
| **半隐式（Symplectic）** | 先显式更新 v，再更新 x | 保能量较好，稳定性介于两者之间；质点-弹簧常用 |

### 5. 质点-弹簧系统仿真（伪代码）

```
初始化 positions[], velocities[], masses[], springs[]

for each frame:
    // 1. 计算力
    for each particle i:
        F[i] = gravity * mass[i]
    for each spring (i,j) with rest length L0, stiffness k:
        d = positions[j] - positions[i]
        F[i] += k * (|d| - L0) * normalize(d)
        F[j] -= ...

    // 2. 积分（Verlet / 半隐式 Euler）
    for each particle i:
        acc = F[i] / mass[i]
        velocities[i] += dt * acc
        positions[i] += dt * velocities[i]

    // 3. 约束（固定点、碰撞等）
    apply_constraints(positions, velocities)

    render(positions)
```

### 7. 前向动力学 vs 逆向动力学

| | 前向动力学（FK） | 逆向动力学（IK） |
|---|----------------|-----------------|
| **已知** | 各关节角度 | 末端位置/姿态 |
| **求** | 末端位置 | 各关节角度 |
| **关系** | 直接链式变换 | 通常非线性，多解或无解，需优化/雅可比迭代 |
| **用途** | 动画关键帧、预览 | 抓取、落脚、视线跟踪 |

**关系**：FK 是 IK 的正过程；IK 为 FK 的逆问题，常用雅可比转置/伪逆、CCD、FABRIK 等求解。

### 8. 蒙皮形变（Linear Blend Skinning，结合示例图）

**作用**：将网格顶点绑定到骨骼，骨骼运动时网格随之自然变形，用于角色动画。

**过程**：

1. **绑定**：为每个顶点 x 分配各骨骼权重 wi（Σ wi = 1）。
2. **骨骼变换**：每根骨骼 i 有变换矩阵 Ti（相对 rest pose）。
3. **顶点变换**：`x' = Σ wi · Ti · x`
4. 图示：顶点 x 受 Bone1、Bone2 影响，分别变到 x'1, x'2，最终 x' 为二者加权混合。

**局限**：大角度旋转时出现“糖果包装”塌陷，可用双四元数蒙皮（DQB）改善。

### 9. 运动捕捉技术路线（≥3 种）

| 路线 | 原理 | 特点 |
|------|------|------|
| **光学式** | 红外相机捕捉反光标记点 | 精度高，需标记点/专用场地 |
| **惯性式（IMU）** | 身体佩戴陀螺仪+加速度计 | 不受遮挡，漂移需校正 |
| **电磁式** | 发射器-接收器电磁场定位 | 实时，受金属干扰 |
| **视觉无标记** | 普通 RGB 相机 + 姿态估计算法 | 低成本，精度受光照/遮挡影响 |
| **机械式** | 关节编码器连体服 | 精确但活动受限 |

### 10. 拉格朗日 vs 欧拉视角的流体仿真

| | 拉格朗日（Lagrangian） | 欧拉（Eulerian） |
|---|----------------------|------------------|
| **跟踪对象** | 跟随**质点/粒子** | 固定**空间网格** |
| **流程** | 粒子携带属性 -> 受力移动 -> 邻域插值 | 网格存储速度/压力 -> 每步更新通量 -> 压力泊松求解 |
| **代表** | SPH、PBF | Stable Fluids、FLIP |
| **优点** | 质量守恒自然，自由表面清晰 | 边界处理简单，压力求解规范 |
| **缺点** | 均匀采样难，需邻域搜索 | 对流扩散耗散，自由面需额外处理（Level Set/粒子辅助） |

**区别**：拉格朗日“跟着物质走”；欧拉“站在格点上看法场变化”。混合方法（FLIP/APIC）结合二者优点。
