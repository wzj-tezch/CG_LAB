## 二、不定项选择题

**1.** 分段函数（\(V_0,V_1\) 为端点）：

- \(0\le t\le 1:\quad \Phi(t)=V_0+\dfrac{t}{3}(V_1-V_0)\)
- \(1\le t\le 2:\quad \Phi(t)=V_0+\dfrac{1}{3}(V_1-V_0)+\dfrac{(t-1)^2}{3}(V_1-V_0)\)

在 \(t=1\) 处位置连续（\(C^0\)），但左导数为 \(\dfrac{1}{3}(V_1-V_0)\)，右导数为 0，**不满足 \(C^1\)/\(G^1\)**。

**答案：A（\(C^0\)）**（亦满足 \(G^0\)，但 \(C^0\) 为最准确选项）

**2.** 4 阶（三次）B 样条，5 个控制顶点，节点向量 \(\{0,0,0,1,2,3,4,4,4\}\)。

对次数 \(k=3\)，定义域为 \([t_k,\, t_{n+1}] = [t_3,\, t_5] = [1,\,3]\)。

**答案：C（\([1,3]\)）**

**3.** 5 个控制顶点、三次 B 样条所需节点数：\(n+k+2 = 4+3+2 = 9\)。

**答案：D（9）**

**4.** 多项式 Bézier **不能精确表示圆**（有理 Bézier 可以）。

**答案：B（圆弧）**

**5.** 三次 B 样条局部支撑宽度为 \(k+1=4\) 个节点区间，改变 \(P_5\) 影响 **4 段**曲线。

**答案：B（4）**

**6.** Bézier 性质：

**答案：C** — 端点处切线方向与特征多边形首末边一致。

- A 错：只过首尾控制点；B 错：过端点（即顶点）；D 错：Bézier 具有全局性，移动一点影响整条曲线。

**7.** 当节点向量取为 **\(\{0,0,0,0,1,1,1,1\}\)**（端点重复 \(k+1\) 次）时，B 样条退化为 **Bézier 曲线**。

**答案：B**

## 三、简答题（以下为全部题目作答）

### 1. 曲面求交方法（≥3 种）

1. **细分求交**：递归细分两曲面参数域，在包围盒相交的子域继续细分直至足够小。
2. **追踪法（Marching）**：从已知交点出发，沿切平面交线方向迭代追踪。
3. **代数法**：将两曲面方程联立，化为多项式方程组用 Gröbner 基或结式求解。
4. **体素/层次包围盒**：用 AABB/OBB 快速排除不相交区域，再精细求交。
5. **隐式-参数混合**：一方隐式 \(F(x,y,z)=0\)，另一方参数化，代入后一维化求解。

### 2. 三次参数样条 vs \(n\) 次 Bézier

| | 三次参数样条 | \(n\) 次 Bézier |
|---|-------------|-----------------|
| 控制 | 控制点 + 端点导数/连续性条件 | \(n+1\) 个控制顶点 |
| 局部性 | 分段三次，**局部**修改 | 整体 \(n\) 次多项式，**全局**影响 |
| 连续性 | 可强制 \(C^2\) 等 | 端点插值，形状由凸包控制 |
| 阶数 | 每段固定 3 次 | 阶数随控制点数升高，高阶不稳定 |

### 3. B 样条控制点共线是否一定为直线？

**正确。** B 样条曲线是控制点的凸组合：\(P(t)=\sum N_{i,k}(t)P_i\)，且 \(N_{i,k}(t)\ge 0\)、\(\sum N_{i,k}=1\)。控制点共线时，\(P(t)\) 恒在该直线上，曲线为直线段（可能参数化重复）。

### 4. 两条二次 Bézier 精确合并为一条的条件

设第一条 \(P_0,P_1,P_2\)，第二条 \(Q_0,Q_1,Q_2\)，且 \(P_2=Q_0\)。

**条件：**

1. **位置连续**：\(P_2 = Q_0\)（已有）；
2. **切向连续（\(C^1\)）**：\(P_2-P_1 = Q_1-Q_0\)（同向且等长）；
3. 此时合并后二次 Bézier 控制点为 \(P_0,\; P_1,\; Q_2\)（中间控制点满足 Bernstein 一致性）。

若仅要求几何合并为一条抛物线，则 \(P_0,P_1,Q_1,Q_2\) 四点共线且参数一致。

### 5. 四次 Bézier 在 \(t=\dfrac{1}{2}\) 处 de Casteljau 计算

控制点：\(P_0(30,0),\; P_1(60,10),\; P_2(80,30),\; P_3(90,60),\; P_4(90,90)\)。

**第一层**（\(t=0.5\)）：

| | 计算 | 结果 |
|---|------|------|
| \(P_0^1\) | \((P_0+P_1)/2\) | (45, 5) |
| \(P_1^1\) | \((P_1+P_2)/2\) | (70, 20) |
| \(P_2^1\) | \((P_2+P_3)/2\) | (85, 45) |
| \(P_3^1\) | \((P_3+P_4)/2\) | (90, 75) |

**第二层**：(57.5, 12.5), (77.5, 32.5), (87.5, 60)

**第三层**：(67.5, 22.5), (82.5, 46.25)

**第四层**：**(75, 34.375)**

```
(30,0) (60,10) (80,30) (90,60) (90,90)
   \  /     \  /     \  /     \  /
 (45,5)  (70,20) (85,45) (90,75)
    \      /       \      /
 (57.5,12.5)(77.5,32.5)(87.5,60)
       \        /
    (67.5,22.5)(82.5,46.25)
          \    /
        (75, 34.375)  ← P(1/2)
```

**答案：曲线在 \(t=\dfrac{1}{2}\) 处的点为 \((75,\; 34.375)\)。**
