## 二、简答题（以下为全部题目作答）

### 1. 平面与球的几何求交

设平面 `n·x + d = 0`（|n| = 1），球心 C，半径 R。

1. 计算球心到平面有符号距离 `h = n·C + d`。
2. 若 |h| > R：不相交；若 |h| = R：相切于一点；若 |h| < R：相交。
3. 交线圆心 `O = C - h·n`（球心在平面上的垂足）。
4. 交圆半径 `r = sqrt(R² - h²)`。
5. 在平面内建立正交基 u, v（u 垂直 v 垂直 n），交线参数化：

```
L(θ) = O + r·cos θ·u + r·sin θ·v,   θ ∈ [0, 2π)
```

### 2. Phong 模型公式含义

```
I = Ia·Ka + Ip·Kd·(L·N) + Ip·Ks·(R·V)^n
```

| 符号 | 含义 |
|------|------|
| I | 观察方向看到的出射光强 |
| Ia | 环境光强度 |
| Ka | 环境光反射系数 |
| Ip | 点光源强度 |
| Kd | 漫反射系数 |
| L | 指向光源的单位向量 |
| N | 表面单位法向量 |
| Ks | 镜面反射系数 |
| R | 理想镜面反射方向 |
| V | 指向视点的单位向量 |
| n | 高光指数（shininess） |

**三项**：环境光、Lambert 漫反射、Phong 镜面高光。

### 3. 多面体 + Phong 的问题与增量式明暗处理

**问题**：多面体用平面片逼近曲面，相邻面片法向突变，导致明暗边界明显、高光形状失真。

**Gouraud Shading**：在顶点算法向/光强，面内**双线性插值**光强。快，但高光模糊、Mach 带。

**Phong Shading**：顶点插值**法向量**，逐像素按 Phong 模型计算。高光准确，计算量更大。

**区别**：Gouraud 插值**光强**；Phong 插值**法向**再算光强。

### 4. 三维图形渲染处理流程

```
几何建模 -> 模型变换(世界坐标) -> 观察变换(相机/视图坐标)
-> 投影(裁剪坐标) -> 视口变换(屏幕坐标) -> 光栅化
-> 逐像素着色(光照/纹理) -> 可见性处理(Z-Buffer) -> 帧缓冲输出
```

### 5. 渲染流水线基本概念

| 概念 | 解释 |
|------|------|
| **裁剪** | 丢弃视锥体外的图元，保留可见部分 |
| **Z-Buffer 消隐** | 每像素维护深度，只保留最近表面 |
| **光栅化** | 将矢量图元转为离散像素片段 |
| **反走样** | 超采样/面积加权等减轻锯齿 |

### 6. 光线追踪递归与加速思路

**伪代码**：

```
Color trace(ray, depth):
    if depth > MAX: return black
    hit = scene.intersect(ray)
    if not hit: return background
    color = ambient + diffuse
    if hit.specular:
        refl = reflect(ray.dir, hit.normal)
        color += ks * trace(Ray(hit.p, refl), depth+1)
    if hit.transparent:
        refr = refract(ray.dir, hit.normal, eta)
        color += kt * trace(Ray(hit.p, refr), depth+1)
    return color
```

**加速思路**：层次包围盒（BVH）减少求交次数；空间剖分（八叉树/均匀网格）只测相关区域物体。

### 7. 层次包围盒与空间剖分

- **BVH**：自顶向下将物体/图元分组，每组用包围盒包裹；射线与根盒不相交则整棵子树跳过。
- **空间剖分**：将空间划分为规则网格或自适应八叉树，每个单元记录所含物体；射线只遍历穿过的单元。

### 8. 八叉树剖分

- 立方体空间递归八等分，直到单元内物体数 ≤ 阈值或达到最大深度。
- **编码**：用路径码（如 3 bit 表示子八分块编号 0–7）标识节点位置。
- **性质**：相邻节点共享面/边/顶点；深度决定空间分辨率；便于快速定位射线经过的叶子节点。

### 9. 光线追踪终止条件

1. 未击中任何物体（命中背景）；
2. 递归深度超过上限；
3. 光强/贡献低于阈值（Russian Roulette 也可用于概率终止）；
4. 射线进入吸收介质且能量耗尽。

### 10. 全局光照 vs 局部光照

| | 局部光照 | 全局光照 |
|---|---------|---------|
| 考虑 | 光源直射 + 环境光近似 | 多次反射/折射、间接光照、色溢 |
| 算法 | Phong、Gouraud | 路径追踪、光子映射、辐射度 |
| 代价 | 低 | 高 |

### 11. BRDF 与 Phong

**BRDF** `fr(ωi, ωo)`：入射方向 ωi 到出射方向 ωo 的反射分布。

**分类**：理想漫反射（Lambert）、理想镜面、Phong/Blinn-Phong（经验模型）、微 facet（Cook-Torrance）、各向异性等。

**Phong 在 BRDF 框架下**：漫反射项 ∝ Kd·(N·L)；镜面项 ∝ Ks·(R·V)^n，非物理（不满足能量守恒/互易性）。

**可逆性修改**：Blinn-Phong 等改为 ∝ (N·H)^n；或归一化分母；Cook-Torrance 等基于物理的 BRDF 天然满足互易性 `fr(ωi, ωo) = fr(ωo, ωi)`。

### 12. 可微渲染 + 深度学习

- **思路**：将渲染过程（光栅化/光线追踪）设计为可微算子，使像素损失对几何、材质、相机参数可求梯度。
- **应用**：逆渲染（从图像重建 3D）、NeRF 优化、纹理/网格直接梯度下降、域随机化生成训练数据。

### 13. 神经辐射场（NeRF）

**组成**：MLP `F(x,y,z,θ,φ) -> (c, σ)` 将位置与视角映射到颜色与体密度。

**渲染**：沿相机射线采样点，用 σ 做体渲染积分（alpha compositing）得到像素颜色；训练时最小化与真实图像的重建误差，优化 MLP 权重。
