# 实验报告：Whitted-Style 光线追踪（Taichi）

## 一、实验目标

1. 理解 Ray Casting 与 Ray Tracing 的本质区别。  
2. 掌握通过次级射线实现硬阴影与理想镜面反射。  
3. 学习将递归光线追踪改写为适合 GPU 的迭代模式。  
4. （选做）实现玻璃折射与全反射处理。  
5. （选做）实现 MSAA 抗锯齿平滑边缘。  

## 二、实验原理

主光线从相机发出，命中场景后执行：

1. **阴影测试**：从交点向点光源发射暗影射线。若中途被其他几何体遮挡，则该点仅保留环境光。  
2. **材质分支**：  
   - 漫反射材质：按 Phong 模型计算颜色并终止该路径；  
   - 理想镜面材质：按反射公式计算方向并继续弹射；  
   - 玻璃材质：按斯涅尔定律计算折射方向；若判别式小于 0，发生全反射。  

反射向量公式：

\[
\mathbf{R} = \mathbf{L}_{in} - 2(\mathbf{L}_{in}\cdot\mathbf{N})\mathbf{N}
\]

折射方向（空气/玻璃介质比 \(\eta\)）：
\[
\mathbf{T} = \eta\mathbf{I} + (\eta\cos\theta_i-\sqrt{k})\mathbf{N},\quad
k = 1-\eta^2(1-\cos^2\theta_i)
\]
当 \(k<0\) 时发生全反射。

## 三、实验环境

- 语言：Python 3.8+
- 框架：Taichi（GPU 优先，失败回退 CPU）
- 文件：`ray_tracing_taichi.py`

## 四、实验内容与实现

### 1. 隐式场景与材质 ID

- Ground Plane：`y=-1.0`，法线 `(0,1,0)`，棋盘格纹理  
- 玻璃球：`(-1.5,0,0), r=1.0, IOR=1.5`  
- 银色镜面球：`(1.5,0,0), r=1.0`  
- 材质标签：`MAT_DIFFUSE=0`, `MAT_MIRROR=1`, `MAT_GLASS=2`

### 2. 迭代式多次弹射

每个像素路径维护：

- `throughput`（初值 1.0）
- `final_color`（初值 0.0）

在固定上限循环中追踪路径，镜面命中继续弹射，漫反射命中累积后终止。该方式规避了 GPU 不擅长递归的问题。

### 3. 硬阴影与精度修复

暗影射线和反射射线都使用偏移起点：

\[
\mathbf{P}_{new}=\mathbf{P}+\epsilon\mathbf{N},\ \epsilon=10^{-4}
\]

成功避免了自相交导致的 Shadow Acne。

### 4. UI 交互

窗口内提供实时滑动条：

- `Light X / Light Y / Light Z`
- `Max Bounces`（1~5，默认 3）
- `MSAA Samples`（1~8，默认 4）

可直观看到阴影移动与反射深度变化。

### 5. 选做一：折射与玻璃材质

将左侧红球替换为玻璃材质，光线命中后不再直接终止，而是继续传播：

- 由法线朝向判断射线是“入射”还是“出射”，并切换折射率比；
- 正常情况下沿折射方向传播；
- 当判别式 \(k<0\) 时触发全反射，改走反射方向。

该实现可观察到玻璃球中的“透视扭曲”与内部反弹现象。

### 6. 选做二：MSAA 抗锯齿

在每个像素内随机发射多条主光线（子像素抖动），将颜色求平均后写回像素。实验中通过 `MSAA Samples` 可实时调整采样数：

- `1`：渲染快但边缘锯齿明显；
- `4`：边缘明显平滑（默认）；
- `8`：边缘最平滑但计算开销更高。

## 五、实验结果与现象

1. 当 `Max Bounces = 1` 时，镜面/玻璃的多次路径信息较少；当 `Max Bounces >= 2` 时二次效果更明显。  
2. 调整光源位置时，地面与球体阴影会实时变化，形成硬边界。  
3. 去掉 `epsilon` 偏移会出现明显黑点噪声；恢复偏移后画面稳定。  
4. 玻璃球可观察到折射造成的背景畸变，特定角度下出现全反射。  
5. `MSAA Samples` 从 `1` 增至 `4/8` 后，球体和棋盘格交界锯齿明显减轻。

## 六、结论

本实验完成了 Whitted-Style 光线追踪核心系统，并进一步实现了两项选做内容：玻璃折射（含全反射）与 MSAA 抗锯齿。结果表明，迭代式弹射结构可在 GPU 上稳定实现多材质路径传播，而多重采样可在可接受开销下显著改善边缘质量。
